    Syntax
    public class SongUnit : UnitBase
    Remarks

    Unit要素（SongUnitの場合はソングトラックの楽譜・調整データ全て、TalkUnitの場合はセリフ単位）を管理します。 シリアライズしたクラスではありません。

    RawRootをはじめとした Raw～ で始まるプロパティは元のxmlのXElementへのアクセスを提供します。XML要素を直接いじる際に使用します。
